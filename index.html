Matchlock Anchor / Go / React

Peer-to-peer sports wagering on Solana

Two people, one match, opposite outcomes. Escrow on-chain, settlement via CPI into TxLINE's oracle program. The winner pays the Solana fee to claim. Loser's funds go to the winner. No house, no spread, no funny business. Built for the Solana + TxLINE hackathon.

  • Anchor program (Rust) with escrow, CPI settlement, pause switch — funds never get trapped by admin action.
  • Go backend (Gin, GORM, Redis, solana-go) serving as keeper + API.
  • React 19 frontend (Vite 8, TypeScript 6, Tailwind 4, Base UI, Zod 4).
