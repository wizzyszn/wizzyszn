# Signal Noir

## North Star: "Precision With Heat"
A sharp engineering portfolio built on a black canvas, near-white content, and a high-energy red signal. The design should feel focused, technical, and confident.

## Colors
- **Signal (`#ff003d`):** Primary action color, active states, section labels, timeline markers, and visual accents.
- **Paper (`#ededed`):** Text, icon strokes, dividers, and subtle grid lines through opacity.
- **Void (`#0a0a0a`):** Page background and the base of every major surface.

## Typography
- **Primary:** Manrope for all headings, body, navigation, and labels.
- Headings are heavy, compact, and direct. Body copy stays calm and readable.
- Letter spacing stays at `0` for a clean, modern feel.

## Elevation
- Surfaces are dark panels with restrained borders.
- Hover states use red outlines and glow only when they indicate interaction.
- The page grid is subtle and structural, not decorative.

## Components
- **Buttons:** Primary = solid red. Secondary = dark surface with pale border and red hover.
- **Cards:** Dark rectangular panels, 8px radius, high-contrast content, red interaction states.
- **Visual panels:** Abstract system graphics using only the core palette.

## Rules
- Use red sparingly enough that it always feels intentional.
- Keep layout dense but breathable; this is a portfolio for scanning real engineering impact.
- Avoid warm beige, brown, blue, purple, and decorative gradients outside the approved palette.
